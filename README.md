## Como contribuir?

* Fork esse repositório;
* Crie uma branch como a sua feature: `git checkout -b my-feature`;
* Commit suas mudanças: `git commit -m 'feat: My new feature'`;
* De um push na sua branch: `git push origin my-feature`;
